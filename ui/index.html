How it works

Serverless AWS Lambda handles every request — no servers to manage, scales to zero.
Storage DynamoDB stores URL mappings with single-digit millisecond reads.
API API Gateway exposes a clean REST interface for creating and resolving links.
Infrastructure Fully managed via Terraform — reproducible, version-controlled, one-command deploy.
